MessageType:
LOGIN = 1
Request server to login

To server:
{
    "type" : 1,
    "body" : 
    {
        "login" : login_string,
        "password" : password_string
    }
}

To client:
{
    "type" : 1,
    "body" :
    {
        "result" : result_bool
        "message" : message_string ( if result True, message empty, because authorization is ok, if result False there is a error messsage)
        "uuid" : str (unique id of player)
    }
}
